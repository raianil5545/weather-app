Git Command and Description

1. git init

Initilaize the git project

2. git status

To check the untracked file

3. git add file name

Add single file in git

4. git add .

Add all files to the git if changes is done

5. git rm --cached filename

To remove git added file from git cached

6. git restore filename

Remove all the new changes if all the changes aren't added

git commit -m "message here"

to commit the files to local repo

7. git remote add origin repo name

To add remote server to the local server

8. git show

To check the file content

9. git push -u origin branchname

Push code the the particular branch in remote server
